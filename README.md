# Art Institute Explorer

A React + TypeScript application for exploring artworks from the Art Institute of Chicago API.

## Features

- Search artworks
- View artwork details
- Save favourites to a personal gallery
- Add notes to saved artworks
- Persistent local storage
- Type-safe validation with Zod

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Zod

## Installation

```bash
npm install
npm run dev
```

## API

This project uses the Art Institute of Chicago API.

https://api.artic.edu/docs/
